В рамках обучения в школе стажеров Иннотех (T1 group) проводилась разработка прототипов витрин данных и непосредственно хранилища

Проект сбора данных и построения историчности для задач DWH (Historical_data_marts)
-Реализован SQL-скрипт построения прототипа объекта-приёмника с историей телефонов-клиентов физлиц.
-Описан алгоритм построения объекта-приёмника в формате ТИС. 
-Сформирован маппинг построения объекта в формате карты отображения S2T. 
-Подготовлены и выполнены бизнес-проверки качества данных полученного прототипа.

Проектирование прототипа DWH (DWH_layers)
-Разработан прототип хранилища данных с поддержкой историчности данных и построением витрин в БД PostgreSQL (перечень и назначение слоев заданы в ТЗ). 
-В качестве источника данных реализован генератор синтетических данных. 
-Версионность и непрерывность данных обеспечиваются по типу SDC2. 
-Процесс ETL обеспечивается с помощью оркестратора Airflow. 
-Тестирование проведено с оркестрацией Airflow. Подготовлены тест-кейсы для проверки обработки различных состояний источника данных. 
-Разработана документация S2T, описание таблиц слоев и витрин.
